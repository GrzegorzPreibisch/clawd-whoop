# Privacy Policy

**clawd-whoop** — Personal WHOOP Data Analysis Tool

**Last updated:** 2026-03-10

## Overview
This application is a personal tool used solely by its developer to access their own WHOOP data for personal health analysis.

## Data Collection
This application does not collect, store, or process data from any third-party users. It accesses only the developer's own WHOOP account data via the WHOOP API.

## Data Sharing
No data is shared with, sold to, or disclosed to any third parties.
